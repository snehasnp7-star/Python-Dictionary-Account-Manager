# Python Dictionary Account Manager

## A menu-driven Python application for managing user credentials using dictionary operations.

## Description

Python Dictionary Account Manager is a simple command-line application that demonstrates the use of Python dictionaries, conditional statements, and user input. The program allows users to view and update a stored username and password through an interactive menu.

This project is intended for beginners who want to practice basic Python concepts while building a functional console application.

---

## Features

- Store user credentials using a Python dictionary
- View the current username
- Update the username
- View the current password
- Update the password
- Handle invalid menu selections

## How It Works

1. The program stores a username and password in a dictionary.
2. A menu is displayed with four options.
3. The user selects an option.
4. The program performs the requested operation.
5. If an invalid option is entered, an appropriate message is displayed.
